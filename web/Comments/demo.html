Comments System Demo

Integration examples pointing to Live Server API

How to Use

To add comments to your page, simply import the Comments class and initialize it on a target container.

<!-- 1. Create a container --> <div id="my-comments"></div> <script type="module"> import { Comments } from './js/Comments.js'; // 2. Initialize the app const app = new Comments(); // 3. Attach to container, set Thread ID, and force Live Mode app.init(document.getElementById('my-comments'), { threadId: 'article-123', apiMode: 'live' // Forces connection to PHP backend }); </script>

Note: The styling is self-contained within the JS, so no external CSS files are required for basic functionality.
