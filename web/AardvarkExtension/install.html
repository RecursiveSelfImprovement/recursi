How to install manually

  1. Click the button above to download the .zip file.
  2. Extract the downloaded .zip file to a folder on your computer.
  3. Open your browser and navigate to chrome://extensions (or edge://extensions if using Edge).
  4. Turn on Developer mode (usually a toggle in the top right corner).
  5. Click the Load unpacked button that appears in the top left.
  6. Select the extracted folder containing the extension files.
  7. You're good to go! Refresh any Aardvark pages to see it in action.
