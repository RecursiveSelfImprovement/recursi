AccuDraw allows the user to make highly optimized use of both keyboard and mouse, working together as a team. It allows the user to keep his or her eyes and mouse where they belong, on the view window. The keyboard is used to both type dimensional information or use any number of keyboard shortcuts to access other functionality. The amount of typing is minimized, as are the mouse motions.

Figure 4: Single letter and two letter shortcut. It is not necessary to press <Enter> following the letter(s) to run the shortcuts. The Shortcuts dialog box does not need to be on screen for shortcuts to be available, but it is useful for learning or for creating your own shortcuts.

The key to making this sort of functionality easy to use is to have lots of feedback. In addition to the compact AccuDraw dialog box, there is a graphic indicator right in the view window showing the origin (also known as the "reference point," typically being the last point placed), and axis rotation. When the user moves the cursor onto one of the axes relative to the origin, a bright line indicates that the point is indexed to that axis. A look at the dialog box will confirm this, showing either X or Y to be zero, or the angle to be either 0, 90, 180 or 270 degrees. Should the user press the <Enter> key while the cursor is indexed, the index will be locked. Again, the dialog box will confirm this, showing the appropriate lock button has been checked.

The dialog box serves as both dynamic coordinate readout and keyboard input. Users will find little need to actually click the mouse on the dialog box, however, since the keyboard focus is dynamically adjusted depending on cursor position, and the text items default to "typeover" mode.
