Editor's Note: This is the conclusion of a two-part series on AccuDraw.

This month, we will take a look at AccuDraw's context sensitivity, usefulness in the 3D environment and customizability. Of course, an entire user's manual would be necessary to cover these topics completely, but the basics are easy to grasp and add greater efficiency and accuracy to MicroStation.

AccuDraw has the ability to make educated assumptions for you based on the active command or previous user input. It

images/figure1_accudraw_settings.jpg

Figure 1: Activate the AccuDraw Settings window from the Settings pulldown.

is these assumptions that make AccuDraw so powerful. This ability is also referred to as context sensitivity. However, there are times when it is more productive to disable this feature.

By default, context sensitivity is on. When it is turned off, AccuDraw is still quite useful, it is just limited. For instance, the compass still functions; however, it will not automatically rotate to

the last data point entered or switch between rectangular and polar mode.

Context Sensitivity is controlled in the AccuDraw Settings window. It is opened from the Settings pulldown (see Figure 1). Shortcut keys also can activate the window. With the AccuDraw window highlighted, press G and then S. This shortcut stands for Go to Settings.

Context Sensitivity is just one option on the AccuDraw Settings window (see Figure 2). It can be turned on or off by pressing the button next to the label.

Let's experiment. Open the accudraw.dgn file we created last month (see "AccuDraw," November 1996 MSM). Or, create a new design file called accudraw.dgn using the seed2D.dgn file as a seed (I suggest not using a production drawing for experimentation). Open the AccuDraw Settings window. Activate AccuDraw and select the Place Line tool. Place a few random lines—horizontal, vertical and diagonal. Now turn Context Sensitivity off and place a few more lines. Notice the differences in the way the compass works. It no longer rotates to the orientation of the last line.

Here is another example. Turn Context Sensitivity on and activate the Place Block tool. Place a data point. Notice that the rectangular coordinate system is now in use. Activate the Place Circle tool and place a data point. The compass has switched to the polar coordinate system. Repeat the same process with Context Sensitivity turned off. The compass will no longer automatically switch between the coordinate systems.
