Level Two

The third dimension

AccuDraw was designed in an effort to simplify the input of 3D data. The major concept is the compass, which deals with all navigation and data input and allows MicroStation elements to be placed in 3D files with ease and accuracy.

Let's take a practical look at AccuDraw's 3D abilities. Create a new file called accu3d.dgn, using seed3d.dgn as a seed file. The 3D design cube should be displayed (see Figure 3). Activate AccuDraw and the Place Line tool. In the isometric view (view 2), snap to the bottom left corner of the cube and enter a data point. The AccuDraw compass should appear.

Now for the beauty of AccuDraw. Press the shortcut key T, for top view. The compass rotates to lie flat on the bottom of the cube, parallel with the top view. Now press F (front) and the compass rotates to align with the front of the cube. Try these other shortcut keys: S (side) and V (view).

With the press of a button the compass can be rotated to align with any view. The shortcut R, Q rotates the compass. This allows the compass to align with any element in the design file so it can be positioned to lie on any plane or element.

With the compass rotated to the desired view, all AccuDraw's 2D concepts come into play. Move the pointer so it snaps to the x or y of the compass. Enter a value in the AccuDraw window—and now you're designing in 3D.

One of the most useful features is the Move Origin (O) shortcut. In the past, it was difficult to identify an origin point that floated in 3D space, one that does not appear at the end of an element or on a grid point. The ability to move the origin of the compass simplifies this function.

Custom AccuDraw

With the defaults delivered with MicroStation, AccuDraw is an impressive tool right out of the box (or should I say straight from the CD?). Yet, there is always room for improvement.

Knowing this, Bentley has built customization features into AccuDraw. The AccuDraw Settings box contains the customization tools at the bottom, under Shortcuts. Unit Roundoff is the first section of the Shortcuts box. As you work with AccuDraw you will notice the last value used is stored in memory. As the pointer nears the distance just used it snaps to that distance. To see this, activate the Place Line tool. Enter a data point, move the pointer to the

right of the compass and notice the x value in the AccuDraw window (for example 54.678). Remember this value and place a second data point.

Now move the pointer to the right again. As the pointer approached the same distance it will snap to the previous value (54.678). This is useful when placing a number of elements at the same distance apart.

Unit Roundoff functions on the same concept, except the value can be specified in the AccuDraw Settings window and will not be replaced with different distances. The value may be overwritten by keyed-in values and snapped tentative points. The Distance option controls linear distance from the compass; Angle controls angles from the compass x and y.

Coordinate System is the second portion of the box. Rotation performs the same function as the T, S, F and other shortcuts discussed earlier. Type switches the compass from rectangular to polar, just as the spacebar shortcut does. If there is a lot of screen space and you do not like shortcuts, these are useful.

Under Operation we encounter some interesting features. The Floating Origin option (on by default) controls whether AccuDraw moves the compass with each data point. It is helpful to turn Floating Origin off if a number of points relative to the same origin point must be entered. The Move Origin shortcut (O) will override this feature.

The Display portion changes the way some of AccuDraw is displayed and contains the Shortcut feature. The red x and green y compass colors can be altered if you wish. Coordinate Readout activates the MicroStation Coordinate Readout box. Nothing different here—standard MicroStation options.
